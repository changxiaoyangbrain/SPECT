# 数据目录

本目录包含项目使用的所有数据文件。

## 📁 目录结构

```
data/
├── input/          # 输入数据（原始数据）
│   ├── Proj.dat   # 投影数据 (128×128×64, float32)
│   └── orbit.xlsx # 轨道参数 (64个角度)
│
└── reference/      # 参考数据（标准答案）
    ├── OSEMReconed.dat  # 参考重建结果 (128×128×128)
    ├── Filtered.dat     # 参考滤波结果 (128×128×128)
    └── *.xif            # 数据元文件
```

## 📝 文件说明

### 输入数据 (input/)

- **Proj.dat**: 原始投影数据
  - 格式：二进制，float32，Little Endian
  - 维度：128 × 128 × 64
  - 说明：第三维对应 64 个采集角度

- **orbit.xlsx**: 采集轨道信息
  - 包含：角度、半径、探头索引等信息
  - 行数：64 行（对应 64 个角度）

### 参考数据 (reference/)

- **OSEMReconed.dat**: 参考重建结果
  - 格式：二进制，float32
  - 维度：128 × 128 × 128
  - 体素尺寸：3.3 mm

- **Filtered.dat**: 参考滤波结果
  - 格式：二进制，float32
  - 维度：128 × 128 × 128
  - 说明：经过高斯滤波（FWHM 10mm）的参考结果

## 🔍 使用方式

数据文件路径在代码中已配置为相对路径，程序会自动从本目录读取数据。

## ⚠️ 注意事项

- 请勿删除或修改输入数据文件
- 参考数据用于评估重建质量，请保持原样
- 如需更新数据，请替换相应文件并保持文件名不变
